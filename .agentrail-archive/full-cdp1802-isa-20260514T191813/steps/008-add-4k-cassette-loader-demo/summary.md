Implemented the 4K cassette-loader demo in sw-cdp1802-io using the shared assembler and emulator. Added a 4K memory map with video at 0x0100, a commented 1802 loader at 0x0000 that uses INP 4 to consume cassette bytes and store them through R1, a cassette demo dropdown entry, progress/status UI, joystick monitor explanatory text, and tests proving CPU-driven cassette consumption and full 256-byte video-page load.
